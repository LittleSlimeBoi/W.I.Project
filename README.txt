Hướng dẫn tái tạo: 
 - Pull source code từ github: https://github.com/LittleSlimeBoi/W.I.Project.git
 - Tải Unity Hub (nếu chưa có): 
   + Trong tab project, Add project pull từ GitHub
   + Tải Unity 2022.3.7f1 (nếu chưa có) ở tab Installs -> Install Editor
 - Mở W.I.P trong trong Unity Hub, giao diện của Unity Editor sẽ được mở, 
ở góc trên bên trái: File -> Build Setting -> Build -> Chọn Path (nên để vào một folder riêng)
 - Sau khi build xong, mở file có tên W.I.Project (hoặc My Application là tên mặc định) để mở dự án

Còn nếu chỉ muốn thử chơi dự án thì truy cập link: https://littleslimeb0i.itch.io/wiproject-wip
